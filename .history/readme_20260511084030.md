# Dreamworlds holdings website - WEDE5020 PART 2

## Project Overview
This is a fully responsive multi-page website built using only HTML5, CSS3, and vanilla JavaScript. It demonstrates semantic markup, responsive layout, and DOM manipulation for WEDE5020 Part 2

## Pages Included

- index.html - Home page with typewriter effect
- about.html - About page with read more toggle
- services.html - Services page with JavaScript price pop-ups
- contact.html - Contact details and location
- enquiry - Enquiry form with JavaScript validation

## Features

- Responsive navigation bar with mobile hamburger menu
- CSS Grid and flexbox layouts
- JavaScript form validation: name, email, service required
- Interactive elements: typewriter, read more, price alerts
- Hover effects and smooth transitions
- Sticky header and consistent footer

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript ES6

## How to Run

1. Download all files into one folder, keeping the folder structure
2. Open index.html in any modern browser
3. Navigate using the menu
4. Test form on enquiry.html - submit with empty fields to see validation

## JavaScript Functionality

1. **Mobile Menu:** Toggles nav on screens < 768px
2. **Typewriter:** Animated h1 text on homepage load
3. **Read More:** Shows/hides extra text on about.html
4. **Price Alerts:** Displays service price from data-price attributes
5. **Form Validation:** Checks required fields and email format using RegEx

## Learning Outcomes
This project demonstrates: HTML for structure, CSS for styling, JS for behaviour
- Separation of concerns:
- DOM selection and event listeners
- Conditional logic and form validation
- Responsive design without frameworks
